home hobbies writing handbook about

the handbook is intended to be a repository of my knowledge and experiences using linux with hyprland, and other fundamental software that i use that don't fit under the hobbies (like neovim and helvum for pipewire, for example)

my hope is that this can act as a handbook for me to refer to when i re-encounter an issue that i have already solved.

a good example of this is the TLP battery manager for lenovo thinkpads, which stores important battery configuration settings in /etc/tlp.conf. i will never remember this location in between the rare occasions that i need to tweak battery settings.

i also hope that this resource will contain some niche information that will be helpful to others encountering similarly obscure problems. my linux journey has been heavily dependent on unexpected saviours and tucked away blogs from 2004; if i can be that resource for someone two decades later then i may just be able to consider the debt repayed :)
